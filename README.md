# MyWishlist

## Informations :

Les documents concernant la répartition des tâches, le processus d'installation se trouvent dans le repertoire documentation.

## Site en ligne :

Le site est en ligne sur https://warplextech.com/MyWishlist/

Des données de bases existent déjà sur le site.

Pour tester le projet, utilisez les identifiants « admin » (qui possède toutes les listes présentes dans la
base de données) et « reserveur » (qui a réservé quelques cadeaux). Ces deux comptes sont
accessibles tout deux sans rentrer de mot de passe.
